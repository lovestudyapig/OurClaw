# 获取某领域下最新研究方向 (searchfile)


## 使用场景
当用户询问具体学术研究领域的最新研究方向时，使用此技能。

## 执行步骤

1. **提取和总结用户目标**：从用户消息的描述中准确提取出用户希望查询的研究领域和关键词，如果用户要求了检索网站需求，请提取相关目标网站。
2. **调用工具获取数据**：调用网络搜索工具确定搜索目标，并进行查询
3. **读取和总结查询所得的多个搜索结果**：为每个关键词进行检索后，生成简洁搜索结果和摘要。摘要需包含 2-3 个段落，字数控制在 500 字以内，需涵盖核心要点。表述需简洁凝练，无需使用完整句子或注重语法规范。该摘要将供他人整合报告使用，因此务必提炼核心信息、剔除无关内容。除摘要本身外，不得添加任何额外评论。
4. **依据检索和摘要结果指定报告大纲**：根据检索和摘要结果，制定报告大纲，明确报告的结构框架与逻辑脉络。
5. **撰写完整报告**：随后基于大纲撰写完整报告，并将其作为最终输出。内容有有结构有段落，最底层的结构不要用1. 或者- 分点描述，直接用文字描述。最终报告需采用 Markdown 格式，内容需详尽深入。字数不少于 2000 字。
